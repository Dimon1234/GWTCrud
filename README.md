# GWTCrud

Cross-browser crud web application implemented in Java language using Spring framework and Google Web Toolkit
==================

## Technologies and frameworks:
* Spring Framework
* Google Web Toolkit
* Hibernate 
* Spring Data JPA
* GWTBootstrap
